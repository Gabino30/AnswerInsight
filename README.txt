AnswerInsight — Project Files
==============================

Contents:
- index.html   -> the website (rename kept as index.html so it works directly on Vercel/Netlify)
- logo.svg     -> vector logo (scalable, use for print/branding)
- logo.png     -> logo with transparent background (use for favicons, social, etc.)

How to use:
1. Extract this folder into: C:\Users\HP-free\Projects\AnswerInsight
2. Double-click index.html to preview it in your browser locally.
3. To publish it online, drag the whole folder into vercel.com or app.netlify.com/drop
4. Before going live, replace the placeholder "#" links on the package buttons
   with your real Stripe Payment Links.
